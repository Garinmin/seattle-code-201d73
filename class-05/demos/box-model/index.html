CSS Box Model

Red is the background color and green is the border color of each paragraph element.

Content - The content of the box, where text and images appear

NOTE: Notice how p elements take up full width of their container

Padding - Clears an area around the content. The padding is transparent

See how the background-color accounts for padding

Border - A border that goes around the padding and content

The background color does not extend beyond the padding

Margin - Clears an area outside the border. The margin is transparent

The box now has 50px of transparent space to the left of bordered box.
